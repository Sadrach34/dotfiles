# Kitty Configuration

> Config path: `.config/kitty/kitty.conf`

## Overview

Kitty is the primary terminal emulator. Config is minimal — focuses on font legibility, slight transparency, powerline tab bar, and cursor animation. Theme colors are managed dynamically by the matugen pipeline.

## Theme Library

150+ themes in `.config/kitty/kitty-themes/`. The active theme is set via:

```
include ./kitty-themes/00-Default.conf
```

To switch themes, change the `include` line to point to any file in `kitty-themes/`. Examples:

| File | Theme |
|------|-------|
| `00-Default.conf` | Default (overridden by skwd-theme.conf) |
| `Dracula.conf` | Dracula dark |
| `Catppuccin-Mocha.conf` | Catppuccin Mocha |
| `Nord.conf` | Nord |
| `Tokyo Night.conf` | Tokyo Night |
| `Gruvbox Dark.conf` | Gruvbox dark |
| `Solarized Dark.conf` | Solarized dark |
| `One Dark.conf` | Atom One Dark |

## Dynamic Color Pipeline

The matugen theming pipeline updates kitty colors automatically:

```
wallpaper → matugen image <path> → skwd-theme.conf regenerated
```

`skwd-theme.conf` is included via the theme system and overrides the base colors in `kitty.conf`. To manually trigger a theme update:

```bash
matugen image ~/.config/hypr/themes/current-wallpaper
```

Or use the Hyprland script:
```bash
~/.config/hypr/scripts/DarkLight.sh
```

The `skwd-theme.conf` file is tracked in git only to capture the current generated state — it should not be manually edited.
